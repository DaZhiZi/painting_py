### 2. bar_line_charts 思路
- 图：
    - 柱形图：
        - 坐标轴：
            
            - x 轴：
                - 箭头（三角形）
                - 刻度
                - 文本信息
            - y 轴：
                -  箭头（三角形）
                - 刻度
                - 文本信息
                
        - rects:
             - rect:
                  - 高度 （temp 温度）
                    - 宽度  （默认 30px）
                    - 颜色 ：
                        - 边缘
                        - 填充
                - 文本信息
                   
    - 折线图：
        - 坐标轴：
            
            - x 轴
                 - 箭头（三角形）
                 - 刻度
                 - 文本信息
             - y 轴：
                 - 箭头（三角形）
                 - 刻度
                 - 文本信息
        - lines:
            
            - line:
                
                - point:
                     - x 坐标
                     - y 坐标
                     - 颜色
                - 文本信息
                - square:
                     - 中心 x 坐标
                     - 中心 y 坐标
                     - 颜色
